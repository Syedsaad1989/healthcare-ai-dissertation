## Research Aim

This dissertation investigates whether explainable classical machine learning models can match or outperform deep learning models for early disease prediction across multiple clinical datasets while maintaining trustworthiness, interpretability, and generalisability.

## Datasets

- UCI Heart Disease
- Pima Indians Diabetes
- Breast Cancer Wisconsin Diagnostic
