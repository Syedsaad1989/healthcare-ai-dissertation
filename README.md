## Project Overview

This dissertation investigates the performance, explainability, and generalisability of classical Machine Learning and Deep Learning models for early disease prediction across multiple clinical datasets.

The project compares traditional machine learning algorithms with a Deep Neural Network while evaluating model interpretability using Explainable Artificial Intelligence (XAI) techniques.

---

## Research Objectives

- Compare the predictive performance of classical Machine Learning models and Deep Learning.
- Evaluate model performance across multiple clinical datasets.
- Assess model explainability using SHAP.
- Investigate the trade-off between predictive performance and interpretability.
- Evaluate model generalisability across different clinical domains.
